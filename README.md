This is a dynamic portfolio application built with Next.js, designed to be fully controlled by an administrator dashboard. It features multi-language support and an integrated blog.

## Features

- **Dynamic Content**: All portfolio content, including projects, skills, and about sections, is managed through an admin dashboard.
- **Admin Dashboard**: A secure interface for administrators to create, update, and delete portfolio items, blog posts, and manage translations.
- **Blog Support**: Integrated blogging platform to share articles and updates.
- **Multi-language Support**: Reach a wider audience with content available in multiple languages.
- **Responsive Design**: Optimized for various devices and screen sizes.
- **SEO Friendly**: Built with Next.js for better search engine optimization.
